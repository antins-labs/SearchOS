# Anti-patterns — deep_web_explorer

## Index
- **反复通用搜索深网结果页** — 搜索引擎索引不到动态生成 (×1, 2026-04-18)
- **只打开首页不交互** — 默认页显示空状态/全集 (×1, 2026-04-18)

## Details

### 反复通用搜索深网结果页
**踩坑**: 代理已到达数据库搜索页，却反复用 `search "site:ecos.fws.gov threatened birds list"` 寻找具体结果页
**原因**: 深网结果是动态生成的，通用搜索引擎索引不到具体查询结果，只能索引入口页；反复搜只会触发 SEARCH_LOOP + 预算耗尽
**改用**: 停止通用搜索，分析当前页面 `<form>` 结构，用 `open` 直接访问带查询参数的 URL（如 `...?status=Threatened&class=Aves`）
**观察**: 1 次  |  **最后**: 2026-04-18

### 只打开首页不交互
**踩坑**: 仅 `open` 数据库首页或通用报告页，期望目标数据直接显示，结果陷入 NO_PROGRESS
**原因**: 数据库默认页通常是空状态或全集，必须通过筛选交互才能拿到特定子集
**改用**: 执行具体交互（click 筛选器、勾选 checkbox）；或从 URL pattern 反推参数后构造带查询参数的 URL 直接访问结果页
**观察**: 1 次  |  **最后**: 2026-04-18
