# 国家级旅游度假区名录 - sxhhy.com 数据提取技能

## 概述

本技能从 www.sxhhy.com 网站提取国家级旅游度假区名录的完整数据集。该名录包含经文化和旅游部认定的所有国家级旅游度假区信息，涵盖2015年第一批至2024年第七批共85个度假区。

## 数据来源

- **网站**: http://www.sxhhy.com/21/9360.html
- **页面标题**: 国家级旅游度假区名录
- **更新日期**: 2024年6月13日

## 数据字段说明

每条度假区记录包含以下字段：

| 字段 | 英文键名 | 说明 | 示例 |
|------|----------|------|------|
| 总序 | overall_id | 总序列号 | "1", "85" |
| 度假区名称 | name | 度假区全称 | "长白山旅游度假区" |
| 所在地 | location | 省份/自治区/直辖市 | "吉林省", "新疆生产建设兵团" |
| 批次 | batch | 认定批次 | "第一批", "第七批" |
| 年度 | year | 认定年份 | "2015", "2024" |
| 序号 | batch_id | 批次内序号 | "1", "22" |

## 功能说明

### list_all - 获取完整列表

返回所有国家级旅游度假区的完整列表。

**示例返回结果**：
```json
{
  "success": true,
  "total": 85,
  "data": [
    {
      "overall_id": "1",
      "name": "长白山旅游度假区",
      "location": "吉林省",
      "batch": "第一批",
      "year": "2015",
      "batch_id": "1"
    },
    ...
  ]
}
```

### get_by_batch - 按批次查询

按认定批次筛选度假区。

**参数**：
- `batch`: 批次名称，支持以下格式：
  - 完整格式："第一批"、"第七批"
  - 简写格式："一"、"七"、"1"、"7"

**示例**：
```json
{
  "success": true,
  "batch": "第七批",
  "total": 22,
  "data": [...]
}
```

### get_by_year - 按年份查询

按认定年份筛选度假区。

**参数**：
- `year`: 年份，如 "2015"、"2024"

**示例**：
```json
{
  "success": true,
  "year": "2024",
  "total": 22,
  "data": [...]
}
```

### get_by_location - 按地区查询

按省份或地区筛选度假区，支持模糊匹配。

**参数**：
- `location`: 省份/地区名称，如：
  - "浙江"（匹配所有浙江的度假区）
  - "广东省"（精确匹配）
  - "自治区"（匹配所有自治区）

**示例**：
```json
{
  "success": true,
  "location": "浙江",
  "total": 8,
  "data": [...]
}
```

### search - 名称搜索

按度假区名称关键词搜索。

**参数**：
- `keyword`: 搜索关键词

**常用搜索示例**：
- "温泉" - 搜索温泉类度假区
- "湖" - 搜索湖泊类度假区
- "雪山" - 搜索雪山类度假区
- "海" - 搜索海滨类度假区

**示例**：
```json
{
  "success": true,
  "keyword": "温泉",
  "total": 6,
  "data": [
    {"name": "汤山温泉旅游度假区", ...},
    {"name": "尧山温泉旅游度假区", ...},
    ...
  ]
}
```

### statistics - 统计数据

返回按批次、年份、地区分组的统计信息。

**示例返回结果**：
```json
{
  "success": true,
  "total_resorts": 85,
  "by_batch": {
    "第一批": 17,
    "第二批": 9,
    "第三批": 4,
    "第四批": 15,
    "第五批": 15,
    "第六批": 3,
    "第七批": 22
  },
  "by_year": {
    "2015": 17,
    "2018": 9,
    "2019": 4,
    "2020": 15,
    "2022": 15,
    "2023": 3,
    "2024": 22
  },
  "by_location": {
    "浙江省": 8,
    "江苏省": 6,
    "山东省": 5,
    ...
  }
}
```

### batches - 批次列表

返回所有批次的列表及每批的基本信息。

**示例返回结果**：
```json
{
  "success": true,
  "total_batches": 7,
  "data": [
    {"batch": "第一批", "year": "2015", "count": 17},
    {"batch": "第二批", "year": "2018", "count": 9},
    ...
  ]
}
```

### locations - 地区列表

返回所有省份/地区的列表及每地区的度假区数量（按数量降序排列）。

**示例返回结果**：
```json
{
  "success": true,
  "total_locations": 30,
  "data": [
    {"location": "浙江省", "count": 8},
    {"location": "江苏省", "count": 6},
    {"location": "山东省", "count": 5},
    ...
  ]
}
```

## 数据覆盖范围

- **时间跨度**: 2015-2024年
- **批次范围**: 第一批至第七批
- **地区覆盖**: 全国30个省级行政区及新疆生产建设兵团
- **总数量**: 85个国家旅游度假区

## 使用注意事项

1. 数据来源于静态网页，更新频率较低，建议定期验证数据时效性
2. 部分度假区名称可能包含标点符号，搜索时建议使用关键词而非全名
3. 地区匹配支持模糊查询，可搜索省级单位的关键词

## 技术实现

- 使用 httpx 直接访问网页，无需浏览器自动化
- 通过 BeautifulSoup 解析 HTML，提取 tab-separated 数据
- 支持代理绕过以适应不同网络环境
- 返回结构化 JSON 数据，便于后续处理和分析
